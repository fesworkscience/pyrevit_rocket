# CPSK Tools - pyRevit Extension

## Быстрый старт

### 1. Установить pyRevit
Скачать и установить: https://github.com/pyrevitlabs/pyRevit/releases

### 2. Скачать репозиторий
```
git clone https://git.gip.su/gip-group/pyrevit_rocket.git
```

### 3. Добавить расширение в pyRevit
- Открыть Revit
- pyRevit → Settings → Custom Extension Directories
- Добавить путь к папке `pyrevit_rocket`
- Перезапустить Revit

### 4. Установить окружение
- Вкладка CPSK → Settings → Окружение
- Нажать "Установить окружение"

Готово!
