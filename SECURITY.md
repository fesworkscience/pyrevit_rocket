# Политика безопасности

## Поддерживаемые версии

| Версия | Поддержка |
| ------ | --------- |
| 1.0.x  | :white_check_mark: |
| < 1.0  | :x: |

## Сообщить об уязвимости

Если вы обнаружили уязвимость безопасности, **не создавайте публичный Issue**.

Вместо этого:

1. Напишите в [Telegram @tehzak2](https://t.me/tehzak2)
2. Или отправьте email на адрес, указанный в профиле

### Что включить в отчёт

- Описание уязвимости
- Шаги для воспроизведения
- Возможные последствия
- Предлагаемое решение (если есть)

### Что ожидать

- Подтверждение получения в течение 48 часов
- Оценка уязвимости в течение 7 дней
- Регулярные обновления о статусе исправления
- Благодарность в релизе (если желаете)

## Общие рекомендации

- Не храните токены авторизации в коде
- Файл `cpsk_settings.yaml` добавлен в `.gitignore`
- Не коммитьте файлы с персональными данными
